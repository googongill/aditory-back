## 프로젝트 소개
> aDitory (어디토리) 는 '링크 자동 분류 웹 서비스' 입니다. <br/>
> 링크를 복사해서 붙여넣기만 하면 자동으로 이를 분석하고 적절한 카테고리로 분류해주는 기능을 제공합니다. <br/>
> 또한, 커뮤니티 기능으로 다른 사람들과 유용한 정보의 교류의 장을 제공합니다. <br/>

<br/>
